本講義では多変量解析の主要な方法を取り上げ，その基本的な考え方を修得することを目的とします．

データを縮約し，その構造をより鮮明に捉えるために少数の変量に変換する手法として回帰分析・主成分分析を，多数の変量を手掛りにデータを分類するための手法として判別分析・クラスタ分析を学びます．
